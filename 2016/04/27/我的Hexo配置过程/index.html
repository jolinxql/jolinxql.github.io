我的Hexo配置过程

install nodejs

讲D:\Program Files\nodejs\node_modules\npm添加到环境变量Path
用cmd(不行的话改用git bash):
$ npm install hexo-cli -g
用cmd(不要用git bash):
$ hexo init e:\Source\Playground\HexoTest # 一定要是一个空文件夹！！！
$ cd e:\Source\Playground\HexoTest
$ npm install
$ hexo server -p 9090
成功了就可以ctrl+C进行下面的步骤

为git添加SSH

然后按照 http://iread.io/2015/09/hexo-guide-3/ 的方法添加ssh
其中，用cmd(不要用git bash):
npm install hexo-deployer-git –save ## 安装部署所需要的插件
按照他的方法继续配置_config.yml，用git bash部署:
$ cd e:\Source\Playground\HexoTest
$ hexo clean && hexo g && hexo deploy && hexo clean
如果deployment失败，试试连接收费地址，还不行就把repository: git@github.com:jolinxql/jolinxql.github.io.git改为
repository: https://github.com/jolinxql/jolinxql.github.io.git 然后delete .deploy_git，重新执行一遍hexo clean && hexo g && hexo deploy && hexo clean（需要密码），然后就可以把repository: https://github.com/jolinxql/jolinxql.github.io.git改为repository: git@github.com:jolinxql/jolinxql.github.io.git（不需要密码）

修改主题：

$ cd themes
$ git clone https://github.com/litten/hexo-theme-yilia.git yilia
(更新，未来有可能做的事情)git bull
修改hexo根目录下的 _config.yml ： theme: yilia

设置头像账户连接等等：

编辑站点主题theme/yilia下的_config.yml的avatar字段（账户字段如facebook,email…同理）：
原则上只要能访问的连接都可设置，如果是本地，可以设置为/img/avatar.jpg(实际地址是/themes/yilia/img/avatar.jpg)
如果是网上可以设置为https://avatars1.githubusercontent.com/u/32269?v=3&s=460
